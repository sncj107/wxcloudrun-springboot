# 微信小程序登录接口使用说明

## 概述

本项目实现了微信小程序的登录功能，遵循微信官方的登录流程。

## 登录流程

```
小程序前端                    后端服务器                   微信服务器
    |                            |                            |
    |-- 1. wx.login() ---------->|                            |
    |                            |                            |
    |<-- 返回 code --------------|                            |
    |                            |                            |
    |-- 2. 发送 code ----------->|                            |
    |                            |                            |
    |                            |-- 3. 请求 code ----------->|
    |                            |                            |
    |                            |<-- 返回 openId/sessionKey -|
    |                            |                            |
    |                            |-- 4. 存储/更新用户信息 ---->|
    |                            |      (数据库)                |
    |                            |                            |
    |<-- 5. 返回登录结果 --------|                            |
    |   (openId + 用户信息)       |                            |
    |                            |                            |
```

## API 接口说明

### 1. 登录接口

**接口地址**: `POST /api/wx/login`

**请求参数**:
```json
{
  "code": "string",        // 必填，wx.login() 获取的临时登录凭证
  "nickname": "string",    // 可选，用户昵称
  "avatarUrl": "string",   // 可选，用户头像 URL
  "gender": 0,            // 可选，性别 0-未知 1-男 2-女
  "language": "string",    // 可选，语言
  "city": "string",        // 可选，城市
  "province": "string",    // 可选，省份
  "country": "string"      // 可选，国家
}
```

**响应示例**:
```json
{
  "code": 0,
  "errorMsg": "",
  "data": {
    "openId": "oXXXX-XXXXXXXXXXXXXXXXX",
    "sessionKey": "XXXXXXXXXXXXXXXXXXXXXXXX",
    "unionId": "oXXXX-XXXXXXXXXXXXXXXXX",
    "userInfo": {
      "nickname": "张三",
      "avatarUrl": "https://...",
      "gender": 1,
      "language": "zh_CN",
      "city": "深圳",
      "province": "广东",
      "country": "中国"
    }
  }
}
```

### 2. 测试接口

**接口地址**: `GET /api/wx/test`

**响应**: 验证服务是否正常运行

### 3. 获取 Access Token

**接口地址**: `GET /api/wx/token`

**用途**: 获取调用其他微信 API 所需的 access_token

## 配置说明

### 环境变量

需要在服务器上配置以下环境变量：

```bash
# 微信小程序配置（必填）
WX_MINIAPP_APPID=你的小程序 AppID
WX_MINIAPP_SECRET=你的小程序 AppSecret

# MySQL 数据库配置（必填）
MYSQL_ADDRESS=你的 MySQL 地址
MYSQL_USERNAME=数据库用户名
MYSQL_PASSWORD=数据库密码
MYSQL_DATABASE=springboot_demo  # 可选，默认 springboot_demo
```

### 获取 AppID 和 AppSecret

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「开发」-> 「基本配置」
3. 获取「开发者 ID」中的 AppID 和 AppSecret

## 小程序前端代码示例

### 基础登录

```javascript
// 1. 调用微信登录
wx.login({
  success: function(res) {
    if (res.code) {
      // 2. 发送 code 到后端
      wx.request({
        url: 'https://your-domain.com/api/wx/login',
        method: 'POST',
        data: {
          code: res.code
        },
        success: function(loginRes) {
          console.log('登录成功', loginRes.data);
          // 保存 openId 等信息
          wx.setStorageSync('openId', loginRes.data.data.openId);
        }
      });
    } else {
      console.error('登录失败:', res.errMsg);
    }
  }
});
```

### 完整登录（包含用户信息）

```javascript
// 注意：从 2021 年 4 月开始，getUserInfo 接口调整为需要用户主动触发
async function loginWithUserInfo() {
  try {
    // 1. 获取 code
    const loginResult = await new Promise((resolve) => {
      wx.login({
        timeout: 10000,
        success: resolve,
        fail: resolve
      });
    });

    if (!loginResult.code) {
      throw new Error('获取 code 失败');
    }

    // 2. 引导用户授权获取用户信息（需要用户点击按钮触发）
    const userInfoResult = await new Promise((resolve, reject) => {
      wx.getUserProfile({
        desc: '用于完善用户资料',
        success: resolve,
        fail: reject
      });
    });

    // 3. 发送到后端
    const response = await fetch('/api/wx/login', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        code: loginResult.code,
        nickname: userInfoResult.userInfo.nickName,
        avatarUrl: userInfoResult.userInfo.avatarUrl,
        gender: userInfoResult.userInfo.gender,
        language: userInfoResult.userInfo.language,
        city: userInfoResult.userInfo.city,
        province: userInfoResult.userInfo.province,
        country: userInfoResult.userInfo.country
      })
    });

    const result = await response.json();
    return result;
  } catch (error) {
    console.error('登录失败:', error);
    throw error;
  }
}
```

## 数据库表结构

用户信息存储在 `Users` 表中，包含以下字段：

- `id`: 主键
- `openId`: 微信用户唯一标识（唯一索引）
- `unionId`: 微信开放平台唯一标识
- `sessionKey`: 会话密钥
- `nickname`: 用户昵称
- `avatarUrl`: 用户头像 URL
- `gender`: 性别
- `language`: 语言
- `city`: 城市
- `province`: 省份
- `country`: 国家
- `createdAt`: 创建时间
- `updatedAt`: 更新时间

## 安全建议

1. **保护 AppSecret**: 不要在客户端代码中暴露 AppSecret
2. **使用 HTTPS**: 生产环境必须使用 HTTPS 协议
3. **Session 管理**: 建议使用 session_key 生成自定义登录态（如 token）
4. **数据加密**: 敏感数据传输时使用微信的加密方案
5. **频率限制**: 对登录接口进行限流，防止恶意调用

## 常见问题

### 1. code 无效或已过期

- code 只能使用一次，有效期为 5 分钟
- 确保每次登录都使用新的 code

### 2. AppID 和 AppSecret 不匹配

- 检查配置的环境变量是否正确
- 确认使用的是当前小程序的 AppID 和 AppSecret

### 3. 无法获取用户信息

- 从 2021 年 4 月起，需要使用 `wx.getUserProfile` 接口
- 必须由用户主动触发（如点击按钮）

### 4. UnionID 为空

- UnionID 只有在开放平台绑定了公众号或移动应用后才会返回
- 小程序单独使用时不会返回 UnionID
